Secara sederhana, NodeJS adalah Javascript versi Server. Pada awalnya, Javascript hanya bisa digunakan pada client saja (browser). Adapun penjelasan lainnya adalah NodeJS merupakan runtime javascript di server. Runtime adalah tempat javascript dapat dieksekusi seluruh kode-nya sehingga dapat menjalankan logika program dan menghasilkan balikan yang diharapkan. Runtime ini pada awalnya hanya berjalan dibrowser saja, sampai Ryan Dahl menginisasi penulisan kode NodeJS pertama pada tahun 2009. Mulai saat itu, Javascript menjadi alternatif bahasa pemrograman untuk mengembangkan aplikasi yang berjalan di server (backend).
